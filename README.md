# InvestIQ - Financial Literacy Dashboard for Students 🚀

[
[
[

**InvestIQ** is a **student-focused financial education platform** designed to teach **16-22 year olds** about smart investing, portfolio management, and financial literacy through an intuitive, modern dashboard interface. Built as a **12-day HTML/CSS coding challenge** during our B.Tech CSE-Data Science coursework at SVIT.

## ✨ **Project Overview**

InvestIQ simulates a **professional investment dashboard** using only **HTML5, CSS3, and responsive design principles**. Perfect for:
- **College students** learning personal finance basics
- **Club presentations** and hackathon demos  
- **Portfolio showcase** for frontend development skills
- **Financial literacy** education without real money risk

**Key Learning Areas:**
- Portfolio overview & asset allocation
- Stock performance tracking (sample data)
- Mutual funds, SIP concepts
- Risk indicators (green/red color coding)
- Currency conversion basics (vs INR)

## 🎯 **Features**

- **📊 Dashboard Overview** - Total value, invested amount, gains/losses
- **💰 Asset Cards** - Stocks, Mutual Funds, Crypto, Fixed Deposits
- **📈 Performance Metrics** - Daily changes, portfolio health
- **💡 Learning Sections** - Beginner-friendly explanations
- **📱 Fully Responsive** - Mobile-first design for student phones
- **🎨 Modern UI** - Dark theme inspired by Zerodha/Groww
- **🔐 Login Mockup** - Professional authentication page design
